## How to use

**N-gram counter**

>   java -jar WordCounter.jar <corpus_path> <output> <ngram> <min_count>

- corpus_path: Path to text file
- output: Path to output file
- ngram: set n-gram size: 1(one-hot), 2(bi-gram), ...
- min-count: Minimum word visible time required in corpus

Example: 
>   java -jar WordCounter.jar all.txt all.1gram.out 1 5
